Frequently Asked Questions

Here are some of the most common questions that we get.
What is the IP address of the Minecraft server? The IP address of the Minecraft server is play.kozers.org
How can I join the Minecraft server? To join the Minecraft server, you need to launch Minecraft and enter the server IP address in the multiplayer menu.
What features does the Minecraft server have? The Minecraft server has a variety of features including custom plugins, mini-games, a player economy, and more.
Is there a whitelist on the Minecraft server? No, the Minecraft server does not have a whitelist. Anyone can join and play.
How can I become part of the active community? To become part of the active community, you can join our Discord (www.kozers.org/discord) server and participate in discussions, events, and more.
